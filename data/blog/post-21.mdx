## Tools & Project

- [AST Explorer](https://astexplorer.net/) (github[源码](https://github.com/fkling/astexplorer))
- [the-super-tiny-compiler](https://github.com/jamiebuilds/the-super-tiny-compiler)
- [JAVASCRIPT AST VISUALIZER](https://resources.jointjs.com/demos/javascript-ast)
- [pegjs](https://github.com/pegjs/pegjs) Parser generator for JavaScript
- [acornjs/acorn](https://github.com/acornjs/acorn) A small, fast, JavaScript-based JavaScript parser
- [acorn-jsx](https://github.com/acornjs/acorn-jsx)
- [Ohm](https://ohmlang.github.io/editor/)  可以可视化的 BNF 编辑器
- [Jison](https://github.com/zaach/jison)  用正则和 BNF 构建通用的 Parser
- [lint-md](https://github.com/lint-md/lint-md) 检查中文 markdown 编写格式规范的命令行工具，基于 AST
- [es-module-lexer](https://github.com/guybedford/es-module-lexer) Low-overhead lexer dedicated to ES module parsing for fast analysis
- [@babel/parser](https://babeljs.io/docs/en/babel-parser) Babel 用来对 JavaScript 语言解析的解析器。
  - [@babel/traverse](https://babeljs.io/docs/en/babel-traverse) 遍历AST
  - [@babel/types](https://babeljs.io/docs/en/babel-types) AST 节点修改
  - [@babel/generator](https://babeljs.io/docs/en/babel-generator) 对新的 AST 进行聚合并生成 JavaScript 代码
  - [@babel/template ](https://babeljs.io/docs/en/babel-template) 封装了基于 AST 的模板能力，可以将字符串代码转换为 AST
